Gerador de Certificados

Páginas:

1. Página principal:
   - Botões: Criar, Editar, Imprimir, Exibir, Excluir e Lixeira.
   - Lista de certificados gerados.

2. Criar:
   - Formulário para inserir informações do certificado.
   - Botão para gerar o certificado.
   - Botão voltar.

3. Editar:
   - Formulário preenchido com as informações do certificado selecionado.
   - Botão para salvar alterações.
   - Botão Voltar.

4. Exibir:
   - Exibição do certificado selecionado para impressão.
   - Botão voltar.

5. Lixeira:
   - Exibição de certificados excluídos.
   - Botões para restaurar e excluir permanentemente certificados.
   - Botão Voltar.

Banco de Dados:

- Banco de dados SQL para armazenar informações dos certificados, como nome, data, evento, etc.
